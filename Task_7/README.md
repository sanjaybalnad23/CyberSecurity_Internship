# C Programming Task 07 - Employee Management System

## Employee Management System

This program is a menu-driven Employee Management System built using C . It allows users to manage employee records, calculate salary statistics, and export employee data to a CSV file.

### Output

#### Add Employee

This option allows the user to add a new employee by entering the employee ID, name, department, designation, and salary.

![Add Employee](screenshots/Screenshot%202026-06-29%20181933.png)

---

#### View Employees

Displays all employee records stored in the system.

![View Employees](screenshots/Screenshot%202026-06-29%20181948.png)

---

#### Search Employee

Searches for an employee using their Employee ID.

![Search Employee](screenshots/Screenshot%202026-06-29%20182024.png)

---

#### Update Employee

Updates the details of an existing employee.

![Update Employee](screenshots/Screenshot%202026-06-29%20182223.png)

---

#### Delete Employee

Deletes an employee record using the Employee ID.

![Delete Employee](screenshots/Screenshot%202026-06-29%20182311.png)

---

#### Salary Statistics

Displays:

* Highest Salary
* Lowest Salary
* Average Salary
* Total Employees

![Salary Statistics](screenshots/Screenshot%202026-06-29%20182740.png)

---

#### Export Employee Data

Exports all employee records to a CSV file.

![Export Employee Data](screenshots/Screenshot%202026-06-29%20182802.png)

---

#### Exported CSV File

Shows the exported employee data stored in the CSV file.

![Exported CSV](screenshots/Screenshot%202026-06-29%20182811.png)

---

#### Exit

Closes the Employee Management System.
