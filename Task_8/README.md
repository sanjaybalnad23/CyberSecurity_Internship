# C Programming Task 08 - Smart File Organizer

## Smart File Organizer

This program automatically organizes files into different folders based on their file extensions. It also provides file statistics, search functionality, duplicate file detection, and generates a text report. (Works on only windows, no linux specific header files are used)

### Output

#### Directory Selection & File Scanning

The user enters the folder path, and the program scans the selected directory to display all detected files.

![Directory Selection & File Scanning](screenshots/Screenshot%202026-06-29%20230843.png)
